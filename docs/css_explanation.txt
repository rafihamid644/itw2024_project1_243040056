===========================================
PENJELASAN DETAIL CSS PORTFOLIO WEBSITE
===========================================

1. GLOBAL STYLES & RESET
-----------------------
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

Penjelasan:
- Reset CSS digunakan untuk menghilangkan default margin dan padding dari browser
- box-sizing: border-box membuat ukuran element dihitung termasuk padding dan border (tidak menambah ukuran total element)
- Ini penting untuk konsistensi tampilan di berbagai browser

2. BODY STYLES
-------------
body {
    min-height: 100vh;
    display: flex;
    flex-direction: column;
    background: linear-gradient();
}

Penjelasan:
- min-height: 100vh memastikan body minimal setinggi viewport (layar)
- display: flex digunakan untuk mengatur layout secara vertikal
- flex-direction: column membuat konten tersusun dari atas ke bawah
- background: linear-gradient() membuat background gradient yang smooth
- overflow-x: hidden mencegah scroll horizontal yang tidak diinginkan

3. NAVIGATION STYLES
-------------------
.navbar {
    position: fixed;
    width: fit-content;
    z-index: 1000;
}

Penjelasan:
- position: fixed membuat navbar tetap di posisinya saat scroll
- width: fit-content membuat lebar navbar sesuai dengan kontennya
- z-index: 1000 memastikan navbar selalu di atas element lain
- transform: translateX(-50%) memposisikan navbar tepat di tengah
- border-radius memberikan sudut melengkung pada navbar

4. HAMBURGER MENU
----------------
.hamburger {
    display: none;
    z-index: 1000;
}

Penjelasan:
- display: none menyembunyikan hamburger di desktop view
- z-index: 1000 memastikan hamburger menu di atas element lain
- transition membuat animasi smooth saat menu dibuka/ditutup
- transform digunakan untuk animasi garis menjadi bentuk X

5. HERO SECTION
--------------
.hero {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

Penjelasan:
- display: flex membuat layout horizontal
- justify-content: space-between memberi jarak antara konten dan gambar
- align-items: center menengahkan konten secara vertikal
- gap: 4rem memberi jarak 4rem antara konten dan gambar
- max-width membatasi lebar konten agar tetap readable

6. JOURNEY SECTION
-----------------
.journey {
    display: grid;
    grid-template-columns: 1fr 1fr;
}

Penjelasan:
- display: grid membuat layout 2 kolom
- grid-template-columns: 1fr 1fr membagi space sama rata
- gap: 4rem memberi jarak antar kolom
- clip-path membuat bentuk polygon yang unik
- position: relative diperlukan untuk pseudo-elements

7. SKILLS SECTION
----------------
.skills-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
}

Penjelasan:
- display: grid untuk layout card berbentuk grid
- repeat(3, 1fr) membuat 3 kolom dengan ukuran sama
- gap memberi jarak antar card
- transition untuk animasi hover
- box-shadow memberi efek floating pada card

8. HOBBIES SECTION
-----------------
.hobbies-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
}

Penjelasan:
- Mirip dengan skills section
- transform: scale() untuk efek hover
- border-radius untuk gambar bulat
- transition untuk animasi yang smooth

9. CONTACT SECTION
-----------------
.contact-form {
    display: flex;
    flex-direction: column;
}

Penjelasan:
- flex-direction: column menyusun form elements vertikal
- gap memberi jarak antar input
- transition untuk animasi focus
- box-shadow memberi depth pada input

10. RESPONSIVE DESIGN
-------------------
@media screen and (max-width: 768px) {
    // Mobile styles
}

Penjelasan:
- Breakpoint 768px untuk tablet dan mobile
- Mengubah layout dari horizontal ke vertikal
- Menyesuaikan ukuran font dan spacing
- Mengaktifkan hamburger menu
- Mengubah grid menjadi single column

11. ANIMATIONS & TRANSITIONS
--------------------------
Penjelasan umum:
- transition: all 0.3s ease - membuat semua perubahan smooth
- transform untuk animasi seperti scale, rotate, translate
- opacity untuk fade effects
- box-shadow untuk depth dan hover effects
- background-gradient untuk warna yang lebih hidup

12. BEST PRACTICES YANG DIGUNAKAN
--------------------------------
1. Mobile-First Approach
   - Desain dimulai dari mobile kemudian desktop
   - Lebih efisien untuk responsive design

2. DRY (Don't Repeat Yourself)
   - Menggunakan variable untuk warna
   - Mengelompokkan style yang sama

3. Performance
   - Menggunakan transform daripada position untuk animasi
   - Menghindari animasi yang berat
   - Optimal loading dengan lazy loading untuk gambar

4. Maintainability
   - Kode terorganisir dengan baik
   - Menggunakan comments untuk dokumentasi
   - Konsisten dalam naming convention

5. Accessibility
   - Kontras warna yang baik
   - Ukuran font yang readable
   - Hover states yang jelas
