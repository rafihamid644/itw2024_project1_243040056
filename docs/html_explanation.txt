1. STRUKTUR DASAR HTML
---------------------
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Portfolio Website</title>
</head>

Penjelasan:
- <!DOCTYPE html> menandakan dokumen HTML5
- lang="en" menentukan bahasa dokumen
- meta charset="UTF-8" untuk encoding karakter universal
- viewport meta tag penting untuk responsive design
- title untuk judul halaman di browser tab

9. CONTACT FORM SECTION
---------------------
The contact form section implements Web3Forms for handling email submissions without a backend server.

Key Components:
1. Form Container
   - Uses class "contact-form" for styling
   - Action points to Web3Forms API endpoint
   - Method set to POST for secure data transmission

2. Hidden Fields
   - access_key: Web3Forms API key for authentication
   - subject: Pre-defined subject for all emails
   - botcheck: Hidden checkbox for spam protection

3. Form Fields
   - Name input (required)
   - Email input (required)
   - Message textarea (required)
   - Result message div for displaying status
   - Submit button

Example Structure:
<form id="contact-form" class="contact-form" action="https://api.web3forms.com/submit" method="POST">
    <input type="hidden" name="access_key" value="[your-api-key]">
    <input type="hidden" name="subject" value="New Portfolio Contact Message">
    <input type="checkbox" name="botcheck" class="hidden">
    
    <div class="form-group">
        <input type="text" id="name" name="name" required>
    </div>
    <!-- Other form fields -->
</form>

Features:
- No backend required
- Built-in spam protection
- Automatic email forwarding
- Success/error message display
- Form validation using HTML5 attributes
